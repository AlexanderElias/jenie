
# CLI

- **Usage**

	- `oxe <operation> [...]`

- **Operations**

- `-v, --version`
- `-h, --help`

- `-c, --compile <input> <output>` Static compile of a project.
	- `-m, --minify`
	- `-c, --comment`

- `-b, --bundle  <input> <output>` Bundles the ES6 modules of o-setup file.
	- `-m, --minify`
	- `-c, --comment`
	- `-n, --name <name>`
