
# CLI

- **Usage**

	- `oxe <operation> [...]`

- **Operations**

- `-v, --version`
- `-h, --help`

- `-c, --compile <input> <output>` Static compiles a project.
	- `-m, --minify`
	- `-c, --comment`
	- `-t, --transpile`

- `-b, --bundle  <input> <output>` Bundles a project.
	- `-m, --minify`
	- `-c, --comment`
	- `-t, --transpile`
	- `-n, --name <name>`
